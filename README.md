# Recommentation_System
Developed a restaurant recommendation system to suggest restaurants based on user preferences.
Used Python and Pandas for data preprocessing, cleaning, and handling missing values.
Applied Label Encoding for categorical features like city and cuisine for memory efficiency.
Scaled numerical features (rating, cost, rating_count) using StandardScaler.
Implemented MiniBatchKMeans clustering to group similar restaurants efficiently.
Added distance-based ranking (Euclidean distance) within clusters for personalized recommendations.
Built a user-friendly Streamlit web application with sidebar inputs for city, cuisine, rating, and cost.
Incorporated robust filtering and fallback mechanisms to handle cases with no exact matches.
Displayed top recommendations dynamically, sorted by distance and rating to ensure relevance.
Achieved a scalable, interactive, and placement-ready project demonstrating real-world recommendation system implementation.
